## Main Session

Press **:** to jump to your main Claude Code session — a persistent tmux
window running `claude` in a directory you choose. The first time you press
**:**, Dispatch prompts for a directory and opens it. After that, **:**
switches straight to the window (recreating it if it was closed).

Use the main session for work that isn't tied to a specific task: exploratory
chats, quick one-off edits, or reviewing context across repos.
